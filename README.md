# Nabz Findz

A luxury e-commerce website for a Nairobi-based beauty and style brand.
Built as a full single-page application — no frameworks, pure HTML/CSS/JS.

<!-- **Live site:** https://YOUR-USERNAME.github.io/nabz-findz -->

## Features

- Midnight dark theme with warm gold editorial aesthetic
- Customer authentication (sign up / sign in / session management)
- Member pricing — 10% discount for logged-in customers
- Wishlist with per-account sync across sessions
- Shopping cart with stock enforcement
- WhatsApp checkout integration
- Admin dashboard (products, orders, customers)
- Image upload per product (base64, stored in localStorage)
- Real-time sold-out updates without page refresh
- Live stock movement chart
- Export orders to CSV and PDF
- Fully responsive — mobile-first design

## Tech Stack

- HTML5, CSS3 (CSS custom properties, Grid, Flexbox)
- Vanilla JavaScript (ES6+)
- Google Fonts: Cormorant Garamond + Outfit
- Data persistence via localStorage
- No build tools, no dependencies, no framework

## Local Development

Just open `index.html` in any browser. No server required.
